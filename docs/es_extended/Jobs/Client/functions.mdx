## ESX.GetJob

Gets the job object data for the specified job name.

```lua
ESX.GetJob(jobName)
```

- jobName: `string`
- return `xJob?`

## ESX.GetJobs

Gets all of the job objects data.

```lua
ESX.GetJobs()
```

- return `<string, xJob>` (array containing instances of xJob)
