**es_extended** is the core resource of the ESX ecosystem.

It handles players and vehicles objects and their related functions.
It also includes multiple pre-defined functions to be used in external resources that utilize ESX.

# Installation

Offical guide to installing **ESX-Overextended**.

:::tip
Make sure you have successfully installed and setup the [database](../guides/database) before continuing!
:::

## Requirements

- [oxmysql<span class="asterisk">\*</span>](https://github.com/overextended/oxmysql/releases/latest/download/oxmysql.zip)
- spawnmanager<span class="asterisk">\*</span>
- [ox_lib<span class="asterisk">\*</span>](https://github.com/overextended/ox_lib/releases/latest/download/ox_lib.zip)
- [illenium-appearance<span class="asterisk">\*</span>](https://github.com/iLLeniumStudios/illenium-appearance/releases/latest/download/illenium-appearance.zip)

## Optional Requirements

These resources aren't required but provide additional functionality.

- [esx_identity](https://github.com/esx-overextended/esx_identity)
- [esx_multicharacter](https://github.com/esx-overextended/esx_multicharacter) (_if using multicharacter, esx_identity **must** be installed_)
- [ox_inventory](https://github.com/overextended/ox_inventory/releases/latest/download/ox_inventory.zip): replaces the default esx's menu-based inventory
- [npwd](https://github.com/project-error/npwd): provides mobile phone functionality in-game

## Install

- Extract the downloaded zip and make sure the extracted folder name is `es_extended` (case sensitive)
- Place it into the `resources` folder in your server.
- Open your database manager/HeidiSQL and import the `esx-overextended.sql` file located in es_extended folder into your database.

- Make sure you are **NOT** using any of these resources:

  - `ox_core`
  - `qb-core`
  - `qbx-core`
  - `essentialmode`
  - `vrp`
  - `basic-gamemode`
  - `mapmanager`
  - `fivem-map-skater`
  - `fivem-map-hipster`
  - `mysql-async`
  - `default_spawnpoint`

- Make sure the start order of the resources in your `server.cfg` looks similar to this:

```diff title="server.cfg"
ensure oxmysql
ensure spawnmanager

ensure ox_lib
ensure es_extended
ensure illenium-appearance
ensure esx_identity        #optional
ensure esx_multicharacter  #optional
ensure ox_inventory        #optional
ensure npwd                #optional
```
