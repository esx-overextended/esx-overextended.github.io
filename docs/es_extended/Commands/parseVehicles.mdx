# Parse Vehicles

This command generates and saves vehicle data for available vehicle models on the client.

```lua
/parsevehicles [processAll]
```

- processAll?: `true` or `false` (include vehicles with existing data (in the event of updated vehicle stats) - defaults to **false**)

_The `Config.EnableDebug` in `es_extended/Config.lua` needs to be **true** in order for this command to work!_

:::info
_By default only superadmins can execute this command_
:::
