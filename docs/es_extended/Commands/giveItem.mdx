# Give Item

This command adds item to the inventory of the player with the specified id.

```lua
/giveitem [id] [itemName] [amount]
```

- id: `number` (the server id of the target player)
- itemName: `string` (the name of the item to add to the specified player)
  - _you can find the list of items in the `items` table of the server's database_
- amount: `number` (the amount of the spcified item to add to the player)

:::info
_By default only admins can execute this command_
:::
