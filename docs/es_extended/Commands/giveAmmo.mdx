# Give Ammo

This command adds ammo to the weapon of the player with the specified id.

```lua
/giveammo [id] [weaponName] [ammo]
```

- id: `number` (the server id of the target player)
- weaponName: `string` (the name of the weapon to add ammo for)
  - _you can find the list of weapons [here](https://wiki.rage.mp/index.php?title=Weapons)_
- ammo: `number` (the amount of ammo for the spcified weapon to add to the player)

:::info
_By default only admins can execute this command_
:::
