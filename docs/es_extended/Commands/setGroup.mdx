# Set Group

This command sets the admin/permission group of the player with the specified server id.

```lua
/setgroup [id] [groupName]
```

- id: `number` (the server id of the target player)
- groupName: `string` (the name of the admin/permission group)
  - _the list of admin groups can be found/modified in `es_extended/Config.lua`_

:::info
_By default only admins can execute this command_
:::
