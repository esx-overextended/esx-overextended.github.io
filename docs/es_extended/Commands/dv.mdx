# Delete Vehicle

This command deletes vehicles in proximity from the game world.

```lua
/dv [radius] [owned]
```

```lua
/cardel [radius] [owned]
```

- radius?: `number` (radius to the player who executes this command to delete vehicles within - defaults to **4.0**)
- owned?: `boolean` (include player-owned vehicles in delete process - defaults to **false**)

:::info
_By default only admins can execute this command_
:::
