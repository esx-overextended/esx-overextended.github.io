# Set Coords

This command teleports the player that executes this command to the coordinates with the specified `x`, `y`, `z`.

```lua
/setcoords [x] [y] [z]
```

- x: `number` (the `x` value of the coordinates)
- y: `number` (the `y` value of the coordinates)
- z: `number` (the `z` value of the coordinates)

_Make sure each argument is seperated with a `space`, not a `comma`!_

:::info
_By default only admins can execute this command_
:::
