# Car

This command spawns a vehicle into the game world.

```lua
/car [model] [owner]
```

- model?: `string` or `number` (the name or hash of the vehicle model - defaults to **[random vehicle]**)
- owner?: `number` (server id of the player to assign vehicle to and save it in database - defaults to **nil**)

:::info
_By default only admins can execute this command_
:::
