## ESX.GetGroup

Gets the group object data for the specified group name.

```lua
ESX.GetGroup(groupName)
```

- groupName: `string`
- return `xGroup?`

## ESX.GetGroups

Gets all of the group objects data.

```lua
ESX.GetGroups()
```

- return `<string, xGroup>` (array containing instances of xGroup)
